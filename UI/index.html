Aloha! Pumble welcomes you! :)


Communicate with PumbleDB by crafting HTTP-GET strings, for example:

Put
http://localhost:3000/put?key=name&value=John%20F.%20Kennedy

Get (display in browser)
http://localhost:3000/get?key=name&Content-Type=text%2Fplain

Get (unspecified Content-Type- download as file)
http://localhost:3000/get?key=name

Delete
http://localhost:3000/delete?key=name


Check out http://fergiemcdowall.github.io/PumbleDB/ for the API and general info!
